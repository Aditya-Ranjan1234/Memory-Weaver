Google Sign-In setup (local)

1. Go to Google Cloud Console -> APIs & Services -> Credentials
2. Configure OAuth consent screen (External is fine for local testing)
3. Create Credentials -> OAuth client ID -> Web application
4. Authorized JavaScript origins:
   - http://127.0.0.1:8000
   - http://localhost:8000
5. Copy the Client ID and set it before running:
   - $env:MW_GOOGLE_CLIENT_ID="YOUR_CLIENT_ID_HERE"

Session cookie signing secret (required):
   - $env:MW_SESSION_SECRET="any-long-random-string"
